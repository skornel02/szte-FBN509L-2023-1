Számolás

Először is számoljuk ki az áramkör eredő áramkörét. A jobb alsó pont azonos a bal alsó ponttal, így a 2. feladat első kérdésére a válasz azonos ezzel az értékkel.
R_ps = R_2 + R_3;
R_ep = 1 / ((1 / R_4) + (1 / R_ps));
R_e = R_1 + R_ep;
A feszültséggel és az eredő ellenállással megállapíthatjuk az áramkörben haladó áram erősségét.
I_1 = U_1 / R_e
I_1 = 3.6000
Az egyenlő a csomóponti törvény alapján -al, amik megkaphatóak Ohm törvénnyel az ellenállás értékek és a rajtuk eső feszültségből.
,
Az megállapítható a következő képlettel:
U_2 = U_1 - (I_1 * R_1)
U_2 = 2.4000
I_2 = U_2 / (R_2 + R_3)
I_2 = 1.2000
I_3 = U_2 / R_4
I_3 = 2.4000
Miután ismerjük az áramerősséget könnyen megállapíthatjuk az feszültséget.
U_4 = R_3 * I_2
